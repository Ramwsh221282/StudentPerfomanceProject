Кафедры - элементы, которые содержат информацию о преподавателях.
Преподавателей внутри кафедр можно закреплять за дисциплинами учебных планов.
Закрепление преподавателей происходит на странице: Учебные планы. В выбранном семестре, выбранной дисциплине.
Преподавателей можно так же зарегистрировать как пользователей. Чтобы преподаватели могли проставлять оценки в контрольных неделях, их необходимо зарегистрировать.
Регистрация преподавателя как пользователя происходит на странице: Пользователи.
Для регистрации необходимо знать реальный email преподавателя. Поскольку после регистрации на этот email отправляется сгенерированный пароль.
Работа с кафедрами (преподавателями) осуществляется на странице: Кафедры.
