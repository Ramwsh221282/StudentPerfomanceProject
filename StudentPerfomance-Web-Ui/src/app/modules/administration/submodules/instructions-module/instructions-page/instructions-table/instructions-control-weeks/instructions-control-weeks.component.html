Контрольные недели на странице администрирования - временные формы, необходимые для того, чтобы в них проставляли оценки. После закрытия контрольной недели на странице администрирования, контрольная неделя сохраняется в отчёт.
Отчёты о контрольных неделях можно просмотреть на странице: Отчёты контрольных недель.
Для создания контрольной недели необходимо:
1. Создать направления подготовки Направления подготовки.
2. Создать учебные планы для направлений подготовки Направления подготовки. Учебные планы.
3. Наполнить учебные планы дисциплинами Учебные планы.
4. Закрепить преподавателей за дисциплинами Кафедры. Учебные планы.
5. Создать студенческие группы и наполнить студентами Студенческие группы.
6. Прикрепить студенческий план и активный семестр студенческим группам. Студенческие группы.
7. Начать контрольную неделю на странице: Контрольные недели.
