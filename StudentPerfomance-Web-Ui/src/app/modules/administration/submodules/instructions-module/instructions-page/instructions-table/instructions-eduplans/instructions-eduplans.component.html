Учебные планы - элементы, которые содержат информацию о семестрах и дисциплинах внутри семестра.
В зависимости от типа направления подготовки, выбранного при создании учебного плана, формируются семестры для учебного плана.
Для бакалавриата - 8 семестров. Для магистратуры - 4 семестра. В каждом семестре учебного плана можно создавать дисциплины. Дисциплины внутри семестра не должны дублироваться по названию.
В каждом семестре учебного плана можно закреплять преподавателя за дисциплиной.
Подготовленные учебные планы в дальнейшем можно прикреплять к студенческим группам.
Для подготовки системы к контрольной неделе необходимо создать учебные планы. 1. В каждом учебном плане необходимо создать ряд дисциплин. 2. В каждом учебном плане необходимо закрепить дисциплины за преподавателями.
Работа с учебными планами направления подготовки осуществляется на странице: Учебные планы.
Работа с кафедрами (преподавателями) осуществляется на странице: Кафедры.
Работа со студенческими группами осуществляется на странице: Студенческие группы.
