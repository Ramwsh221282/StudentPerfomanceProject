Студенческие группы - элементы, которые содержат информацию о студентах.
Для того, чтобы студенческая группа смогла участвовать в контрольной неделе необходимо: 1. Добавить студентов в студенческую группу. 2. Закрепить учебные план и назначить активный (текущий) семестр студенческой группе. 3. Убедиться, что, в назначенном активном семестре назначенного учебного плана присутствуют дисциплины. 4. Убедиться, что, в назначенном активном семестре назначенного учебного плана дисциплины имеют преподавателей.
В зависимости от типа направления подготовки, выбранного при создании учебного плана, формируются семестры для учебного плана.
Работа с учебными планами направления подготовки осуществляется на странице: Учебные планы.
Работа со студенческими группами осуществляется на странице: Студенческие группы.
