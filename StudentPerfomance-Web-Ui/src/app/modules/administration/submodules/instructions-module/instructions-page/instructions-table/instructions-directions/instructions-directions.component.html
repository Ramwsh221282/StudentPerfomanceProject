Направления подготовки - корневой элемент, которому принадлежат учебные планы.
Один экземпляр направления подготовки может включать разные учебные планы.
Учебные планы внутри направления подготовки не должны дублироваться по годам.
Для подготовки системы к контрольной неделе необходимо создать направления подготовки.
Работа с направлениями подготовки осуществляется на странице: Направления подготовки.
Работа с учебными планами направления подготовки осуществляется на странице: Учебные планы.
