Информация о взаимодействии с кафедрами.
Кафедра.

Кафедра объявляет в себе название кафедры и содержит преподавателей.

Преподаватели нужны для того, чтобы можно было зарегистрировать пользователя как преподавателя.

В текущей системе один преподаватель содержится в одной кафедре.

Кафедра. Добавление.

При добавлении кафедры в систему, необходимо учитывать её уникальное название.

Кафедра. Удаление.

При удалении кафедры, удаляются и её преподаватели.

Пользователи, которые были зарегистрированы как преподаватели удаленной кафедры - не удаляются.

Кафедра. Редактирование.

Редактирование кафедры допускает изменение названия кафедры.

Кафедра. Преподаватели.

Операции над преподавателями в кафедрах:

Добавление преподавателя.

Удаление преподавателя.

При удалении преподавателя, пользователь, зарегистрированный как этот преподаватель - не удаляется.

Редактирование данных преподавателя.

При редактировании преподавателя, пользователь, зарегистрированный как этот преподаватель - не изменяется.
