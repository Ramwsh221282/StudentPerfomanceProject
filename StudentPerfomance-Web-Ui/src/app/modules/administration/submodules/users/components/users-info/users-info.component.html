Пользователи.

В системе допускаются два вида пользователей - администратор и преподаватель.

Администраторов можно свободно добавлять в систему. Они будут иметь права на изменение данных в системе (например дисциплин).

Преподавателей можно добавить только, если есть создана запись преподавателя в кафедре.

При регистрации пользователей указывается их электронная почта.

Электронная почта пользователя всегда должна быть уникальна.

При регистрации пользователя на его почту высылается информация о пароле для входа.

Пользователи авторизуются используя свои почту и пароль, которые находятся в этой системе.

При удалении пользователя на его почту высылается информация о том, что его учетная запись была удалена.

Закрывать контрольную неделю для формирования отчёта.
