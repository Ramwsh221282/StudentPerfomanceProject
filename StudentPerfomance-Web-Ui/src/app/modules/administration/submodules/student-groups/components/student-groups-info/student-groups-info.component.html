Информация о взаимодействии со студенческими группами.
Студенческие группы.

Студенческие группы объединяют в себе название группы, учебный план этой группы, и текущий активный семестр группы.

Студенческие группы содержат в себе студентов.

В студенческую группу можно добавлять студентов.

Студенческой группе можно задавать учебный план.

Название студенческой группы должно быть всегда уникальным.

Студенческая группа. Добавление.

При добавлении студенческой группы необходимо учитывать уникальность названия учебной группы.

Студенческая группа. Редактирование.

Редактирование студенческой группы позволяет:

Изменять название студенческой группы.

Изменять учебный план и текущий семестр учебной группы.

Смешивать учебные группы между собой (объединять в одну).

Изменение активного семестра происходит при изменении учебного плана группы.

При изменении названия учебной группы необходимо учитывать уникальность нового названия.

Студенческая группа. Удаление.

При удалении студенческой группы удаляются и её студенты.

При удалении студенческой группы из учебного плана пропадает запись о том закрепленной учебной группе.

Студенческая группа. Студенты.

В студенческую группу можно добавлять студентов.

При добавлении студента в студенческую группу необходимо учитывать уникальность его зачетной книжки в системе в целом.

Студенты в студенческой группы могут иметь статус "Активен" и "Неактивен".

Активные студенты учитываются в контрольной неделе.

Неактивные студенты в контрольной неделе участие не принимают.
