Информация о взаимодействии с контрольными неделями.
Контрольные недели.

Контрольные недели создаются для конечно формирования отчёта за контрольную неделю.

Контрольные недели, созданные в панели администрации являются временным местом, где преподаватели проставляют оценки.

Только преподаватели изменяют содержимое контрольной недели.

При закрытии контрольной недели, запись о контрольной недели в панели администрирования удаляется и формируется отчёт.

В панели администрирования отображаются только активные контрольные недели. Закрытые контрольные неделю преобразуются в отчёт.

Контрольные недели. Добавление.

Добавление контрольной недели создает открытую сессию контрольной недели для взаимодействия.

Преподаватели могут проставлять оценки только в текущей открытой контрольной недели.

Контрольную неделю нельзя начать если:

Хотя бы одна группа не имеет активного учебного плана:

Хотя бы одна группа не имеет активного семестра учебного плана:

Хотя бы одна дисциплина в активном семестре группы не имеет назначенного преподавателя:

Контрольные недели. Закрытие.

Закрытие контрольной недели закрывает контрольную неделю и формирует отчёт.

При закрытии контрольной недели запись активной контрольной недели удаляется.

При закрытии контрольной недели преподаватели больше не могут никак повлиять на её содержимое

При закрытии контрольной недели учебная группа МОЖЕТ переводится на следующий семестр. Если это была вторая контрольная неделя группы, то эта группы переводится на следующий семестр.

При переводе группы на следующий семестр после закрытия контрольной недели, убедитесь, что в следующем семестре группы имеются дисциплины.
