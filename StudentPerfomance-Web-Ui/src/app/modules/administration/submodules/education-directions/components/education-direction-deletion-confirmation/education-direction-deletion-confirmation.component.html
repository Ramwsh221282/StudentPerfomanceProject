Подтверждение

При удалении направления подготовки будет удалено следующее:

1. Направление подготовки.

2. Учебный план связанного направления.

3. Студенческие группы связанного плана.

4. Семестры связанного плана.

5. Дисциплины связанных семестров.

6. Студенты связанной группы.

7. Оценки студентов.
