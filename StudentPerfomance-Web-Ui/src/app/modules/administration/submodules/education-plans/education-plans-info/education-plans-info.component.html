Информация о взаимодействии с учебными планами.
Учебные планы.

Учебные планы объединяют в себе информацию о годе учебного плана, для какого направления подготовки сформирован.

Год учебного плана для одного направления подготовки должен всегда быть уникальным.

Учебные планы так же содержат информацию о семестрах учебного плана.

При создании учебного плана для магистратуры формируются 4 пустых семестра.

При создании учебного плана для бакалавриата формируются 8 пустых семестра.

Чтобы просмотреть семестры учебного плана, необходимо нажать на кнопку семестра соответствующего учебного плана.

В меню семестров можно просмотреть меню дисциплин семестров.

Управление дисциплинами семестров. Добавление.

Каждый семестр может содержать в себе N количество дисциплин.

Каждая дисциплина в семестре должна быть уникальна (без повторяющихся названий).

Управление дисциплинами семестров. Редактирование.

При редактировании дисциплины допустимы следующие операции: закрепление преподавателя, изменение название дисциплины.

При закреплении дисциплины преподавателю, преподавателю добавляется дисциплина.

Преподаватель не может вести одну и ту же дисциплину, за исключением, если они из разных семестров.

Например, преподавателю допускается ведение дисциплины "программирование" из 1 семестра, и ведение дисциплины "программирование" из 3 семестра.

При редактировании названия дисциплины необходимо учитывать уникальность нового имени.

Управление дисциплинами семестров. Удаление

При удалении дисциплины в семестре удаляется сама дисциплина.

Если в момент удаления у дисциплины закреплен преподаватель - то данная дисциплина открепляется от преподавателя.

Управление семестрами.

Семестры невозможно добавить или удалить, только изменять дисциплины внутри семестров разрешено.
