Информация о взаимодействии с направлениями подготовок
Направление подготовки

Направление подготовки объявляет тип направления (бакалавриат и магистратура), код и наименование.

Направление подготовки объединяет в себе учебные планы, созданные для направления.

Направление подготовки не может содержать в себе учебные планы одинакового года.

Добавление направления подготовки

Добавление направления подготовки - создаёт экземпляр направления подготовки в системе.

Код направления подготовки должен быть уникальным.

Наименование и тип направления подготовки не могут быть пустыми.

Удаление направления подготовки

Удаление направления подготовки - удаляет направление подготовки из системы.

При удалении направления подготовки удаляется следующее:

Учебный план, привязанный к направлению подготовки:

Семестры связанного учебного плана:

Дисциплины семестров связанного учебного плана

Редактирование направлений подготовки

Редактирование направлений подготовки позволяет изменить код, наименование и тип.

При изменении кода направления подготовки, новый код должен быть уникальным.

Фильтрация направлений подготовки

Фильтрация списка позволяет отфильтровать список направлений подготовки по заполненным критериям.

При фильтре допустимо не заполнять все поля, а заполнить только необходимые (например только код или тип).
